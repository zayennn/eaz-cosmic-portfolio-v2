## 🌌 Overview

**EAZ Cosmic Portfolio v2** is more than just a portfolio — it's an **interactive cosmic journey**. Built from scratch without any CSS frameworks, every component is handcrafted to create a unique, immersive experience that feels like exploring a digital galaxy.

### ✨ What Makes This Different?

This isn't a template. This isn't a clone. Every pixel, every animation, every interaction is **custom-designed and custom-coded**. The goal? Make visitors say **"Wow, I've never seen anything like this before"** the moment they land on the page.

---

## 🎨 Inspiration & Evolution

This portfolio is the culmination of two previous projects:

| Version | URL | Focus |

| **Cosmic Portfolio v2** | [portfolio-cosmic-v2.great-site.net/](https://portfolio-cosmic-v2.great-site.net/) | **The best of both worlds combined** |

I took the immersive cosmic aesthetic from v1 and the clean, modern structure from Fix Portfolio, then leveled everything up with **Laravel 10**, **GSAP animations**, and **fully interactive components**.

### No CSS Frameworks
This project intentionally avoids Bootstrap, Tailwind, or any CSS framework. All styles are **handcrafted custom CSS** for complete creative control.

### No JavaScript Frameworks
Frontend interactivity uses **vanilla JavaScript** (no React, Vue, or Alpine) combined with **GSAP** for advanced animations and smooth scrolling.

---

## 🎯 Key Design Principles

### 1. **First-Time Viewer Experience**
Every page is designed so a first-time visitor sees something they've **never seen before**:
- Skills as an interactive constellation, not boring cards
- Projects orbiting a sun, not a static grid
- Interests on a pannable star map, not a list
- Contact form as a holographic terminal, not plain inputs

### 2. **No Templates, No Shortcuts**
- Zero CSS frameworks — pure custom CSS
- Zero JavaScript frameworks — vanilla JS + GSAP
- Every animation is handcrafted
- Every layout is original

### 3. **Cosmic Consistency**
The entire site maintains a consistent cosmic/space theme:
- Dark backgrounds with star fields
- Glowing accents (purple, blue, cyan)
- Orbital/gravitational metaphors
- Futuristic, holographic aesthetics

## 🚀 Getting Started

### Prerequisites
- PHP 8.1+
- Composer
- Laravel 10

### Installation

```bash
# Clone the repository
git clone https://github.com/zayennn/eaz-cosmic-portfolio-v2.git

# Navigate to project folder
cd eaz-cosmic-portfolio-v2

# Install dependencies
composer install

# Copy environment file
cp .env.example .env

# Generate application key
php artisan key:generate

# Configure email in .env (for contact form)
# MAIL_MAILER=smtp
# MAIL_HOST=smtp.gmail.com
# MAIL_PORT=587
# MAIL_USERNAME=your-email@gmail.com
# MAIL_PASSWORD=your-app-password
# MAIL_ENCRYPTION=tls
# MAIL_FROM_ADDRESS=your-email@gmail.com

# Start the development server
php artisan serve
